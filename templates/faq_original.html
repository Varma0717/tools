Getting Started

How do I create a free account?

Creating a free account is simple. Click "Start Free" on our homepage, enter your email address and create a password. You'll have instant access to 100+ SEO tools without any credit card required. Verify your email to unlock additional features.

What tools are available in the free plan?

The free plan includes over 100 SEO tools including Meta Tag Analyzer, SSL Checker, Robots.txt Validator, Basic Keyword Research, Website Speed Test, Broken Link Checker, and many more. You can perform up to 50 analyses per day with the free plan.

How do I run my first SEO analysis?

After logging in, go to the Tools section and select any tool you'd like to use. Enter your website URL or the required information, and click "Analyze". Most tools provide results within 30 seconds. Start with the Website Audit tool for a comprehensive overview.

Tools & Features

What's the difference between free and Pro tools?

Free tools provide basic analysis with daily limits. Pro tools offer unlimited usage, advanced features like competitor analysis, backlink monitoring, rank tracking, white-label reports, API access, and priority support. Pro users also get more detailed reports and data export options.

How accurate are the SEO analysis results?

Our tools use industry-standard algorithms and real-time data from multiple sources to ensure high accuracy. We regularly update our analysis engines to reflect the latest search engine guidelines and best practices. Results are typically 95%+ accurate for technical SEO factors.

Can I analyze competitor websites?

Yes! With a Pro account, you can analyze any public website including competitors. Use our Competitor Analysis tools to compare SEO metrics, discover their top keywords, analyze their backlink profiles, and identify opportunities to improve your own SEO strategy.

How do I export analysis reports?

Pro users can export reports in PDF, CSV, and Excel formats. After running an analysis, click the "Export" button and choose your preferred format. Reports include all analysis data, recommendations, and can be white-labeled for client presentations.

Technical Support

Why is a tool showing "Error" or not working?

Tool errors can occur due to website accessibility issues, invalid URLs, or temporary server problems. First, check that your URL is correct and publicly accessible. If the issue persists, try again in a few minutes or contact support with the specific error message.

How fast should analysis results appear?

Most tools provide results within 10-30 seconds. More comprehensive analyses like full website audits may take 1-2 minutes. If an analysis takes longer than 5 minutes, there may be an issue - try refreshing the page or running the analysis again.

Can I analyze password-protected websites?

Our tools can only analyze publicly accessible websites. Password-protected or private websites cannot be analyzed. However, you can analyze specific pages if you have public URLs, or use our API with authentication credentials for custom integrations.

SEO Best Practices

How often should I run SEO analyses?

For ongoing monitoring, run basic analyses weekly and comprehensive audits monthly. After making significant website changes, run immediate analyses to check impact. Pro users can set up automated monitoring and alerts for key metrics like rankings and backlinks.

What are the most important SEO factors to focus on?

Focus on technical SEO (site speed, mobile-friendliness, crawlability), content quality and relevance, proper meta tags, internal linking structure, and building quality backlinks. Our Website Audit tool prioritizes issues by impact to help you focus on the most important improvements first.

How long does it take to see SEO improvements?

Technical fixes can show results within days, while content and link-building efforts typically take 3-6 months to show significant impact. Track progress with our rank monitoring tools and focus on consistent, quality improvements rather than quick fixes.
